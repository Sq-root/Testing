What is OOPM?

Object-oriented programming (OOP) is a programming paradigm based on the concept of "objects", which can contain data, in the form of fields (often known as attributes or properties), and code, in the form of procedures. Object-oriented programming aims to implement real-world entities like inheritance, hiding, polymorphism etc in programming.The main aim of OOP is to bind together the data and the functions that operate on them so that no other part of the code can access this data except that function.
